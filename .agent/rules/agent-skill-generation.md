# Agent Skill Generation Rule

タスクが完了、または一段落した際には、その過程で得られた再利用可能な知見や技術的なベストプラクティスを整理し、**Agent
Skill を作成または更新すること**。

## 手順

1. **知見の抽出**: 修正内容、トラブルシューティングの結果、ライブラリの特有の挙動などを振り返る。
2. **スキルの作成/更新**: `.agent/skills/`
   ディレクトリ配下に、skill-creatorスキルを用いてスキルを作成する
3. **内容例**:
   - 名前と説明 (Frontmatter)
   - 症状や背景
   - 具体的な解決策（コード例、設定例）
   - 注意事項や制約
